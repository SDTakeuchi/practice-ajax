
=======================
Django Ajax Sample
=======================

This is a Django Ajax Sample project.


Requirement
===========

:Python: 3.7.0
:Django: 2.2.3
:django-bootstrap4: 0.0.8
:pytz: 2019.1
:sqlparse: 0.3.0

Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/ajax

2. migrate model::

    python manage.py migrate

3. create superuser::

    python manage.py createsuperuser

4. start server::

    python manage.py runserver
